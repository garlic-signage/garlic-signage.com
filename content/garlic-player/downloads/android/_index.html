Garlic Digital Signage Player runs on a wide range of Android devices such as TV boxes, tablets, or FireTV sticks. Garlic-Player for Android can be installed in just a few steps. Before downloading, please check the system requirements and follow the installation notes below.

Android Downloads

Here you can download Garlic-Player for Android in different builds. All packages are provided directly as APK files and can be installed on compatible devices.

Requirements

  • Android 7.0 or higher
  • Recommended: 2 GB RAM, stable Wi-Fi or LAN connection

Installation

  1. Download the APK below.
  2. Enable “Install from Unknown Sources” in your Android settings.
  3. Open the APK and follow the installation prompt.

Releases

The stable release is recommended for production environments, digital signage networks in daily use, and long-term installations. Developer builds are intended for testers and advanced users who want to try new features earlier.

Use the stable build if you need reliability. Use the developer builds if you want to follow the latest development and don’t mind the occasional crash.

Most users should start with the stable release. Switching to a developer build is only suggested if you need a specific new feature
