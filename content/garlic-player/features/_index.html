Garlic Player is more than a media player. It’s an open-source solution built for professional digital signage. Explore the features that give you full control.

Core Capabilities

🌐SMIL Compatibility

Use the open SMIL standard to precisely control complex multi-zone layouts and playlists without proprietary software.

💻True Cross-Platform Support

Garlic-Player runs smoothly on Windows, Linux, and macOS. It's built for deployment on popular hardware like Raspberry Pi and other single-board computers.

🔒No Vendor Lock-in

Garlic Player is 100% open source. Integrate it with any SMIL-compatible management software and avoid being tied to a specific vendor.

🚀Advanced Playback Control

Control content with triggers, media synchronization, and the REST API. Take remote control and capture screenshots for easy maintenance.

📺Flexible Multi-Zone Layouts

Split your screen into multiple zones and show different content simultaneously. Create dynamic layouts for news tickers, widgets, and more.

📊Seamless Reporting

Get real-time status and playback data directly via WebDAV for easy integration into your monitoring tools.

⬇️Reliable File Downloads

Files are transferred securely with resumed download functionality. This ensures uninterrupted content delivery, even with network issues.

📡Remote Management

Manage and configure your players remotely via the command line or a REST API. This is ideal for large-scale rollouts and automated deployments.

Technical Specifications

Supported Content Formats

  • Images: JPG, GIF, PNG, SVG
  • Videos: H264/265, VC1, VP8/9, MPEG-1/2/4
  • Audio: WAV, WMA, MP3, OGG/Vorbis
  • Web: HTML5, CSS, and W3C Widgets
