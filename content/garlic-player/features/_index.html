Digital Signage Features

  • Trigger based on time, touch/click events, access keys and media synchronisations
  • Multi monitor support
  • Multiple overlapping zones for split screens, news ticker etc…
  • Remote Configuration and manually
  • WebDav Reporting Play-logs, Inventory…
  • Resume Downloads
  • Video codecs H264/265, VC1, VP8/9, MPEG-1/2/4…
  • Audio WAV, WMA, MP3, OGG/Vorbis
  • Images: JPG, GIF, PNG, SVG 1.2 Tiny
  • Playlist format W3C SMIL
  • Web format HTML5 and W3C Widgets
  • SMIL sync-base and event support
  • Rest-API with screenshot and network trigger

Requirements

  • Android
  • Linux
  • macOS
  • Windows
