Key Features

⚡Automatic Player Registration

SMIL players connect and register automatically, minimizing setup time and configuration effort.

📑SMIL Index Management

Serve SMIL indexes directly to internal players and download them seamlessly from your CMS.

🔎Lightweight SMIL Parser

Efficient parsing of SMIL indexes ensures fast performance even on small devices.

⬇️Optimized Media Downloads

Uses hashtags to avoid redundant downloads, saving bandwidth and storage space.

🌐Simple Web Interface

A minimal web GUI allows you to manually refresh indexes with just one click.

✅Tested & Reliable

Full unit test coverage ensures stability, reliability, and consistent performance.

Requirements

💻PHP 5.6+

Fully tested with PHP 7.x for compatibility and performance.

🔗php-curl

Required for handling network requests and content delivery.

🖥️Web Server

Works with Apache, nginx, or other standard web servers.

🗂️CMS Caching

For caching efficiency, the digital signage CMS should generate MD5 checksum files.
