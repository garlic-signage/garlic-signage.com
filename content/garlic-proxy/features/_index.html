--- title: "The Features | Garlic-Proxy" pageHeader: "The Features" smallTitle: "Features" description: "Learn about the features of garlic-proxy ✓ automatic registration ✓ web gui ✓ redundant downloads preventions ✓" ---
  • SMIL Player automatic registration
  • serve SMIL-Indexes to internal player
  • download SMIL-Indexes from a CMS
  • small SMIL parser
  • media download with hashtag to prevent redundant downloads
  • small web GUI for manually index refresh
  • full unit test coverage

Requirements

  • PHP 5.6+ (Tested with 7.x)
  • php-curl
  • Apache, nginx, or other web servers
  • for caching capabilities, Digital Signage CMS should create md5 files
