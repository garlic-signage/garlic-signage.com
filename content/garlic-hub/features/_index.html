Garlic-Hub is currently in active development and is intended for evaluation and testing. We recommend closely monitoring its progress and providing feedback before using it in a production environment.

  • Authentication

    Session-based login with remember-me functionality and basic OAuth2 token authorization

  • Media Management

    Hierarchical content organization with multi-source uploads (local, external links, screencasts, camera, stock platforms with API-key)

  • Playlist Management

    Graphic display zone editor for multi-zone playlists

    Trigger which is based on time, touch, key, and network messages

    Conditional play: decide when this content should be played

    Playlist management of video, audio, images, Widgets, and export in industry-standard SMIL format.

  • Internationalization

    English, German, French, Spanish, Greek, Russian. More language will come.
